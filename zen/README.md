# Zen Commands

```
Ctrl + Option + C  = Display/Hide compact mode
Ctrl + Option + S  = show sidebar
Ctrl + Shift + S = save page snippet
Ctrl + Shift + D = New bookmark
Shift + Option + D = Enable/Disable Dark Reader (Extension)
Ctrl + Shift + Y = Open Vaultwarden extension
Ctrl + Period = Open 1Password extension
Ctrl + Option + Q = Switch between workspaces

Ctrl + 1-9 = Change to tab 1-9

Ctrl + -> = Forward
Ctrl + <- = Backwards

Ctrl + Option + V = Vertical Split
Ctrl + Option + H = Horizontal Split
Ctrl + Optino + U = Unsplit view
```
